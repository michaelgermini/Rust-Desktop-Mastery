   - Pipeline automatisé

## Objectifs d'apprentissage

À la fin de ce chapitre, vous serez capable de :

- Transformer des maquettes SVG en code Rust
- Automatiser le workflow design → code
- Maintenir la cohérence entre design et implémentation
