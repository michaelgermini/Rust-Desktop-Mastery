# Chapitre 11 : Composants Réutilisables

Ce chapitre présente la création de composants UI réutilisables basés sur votre design system.

## Sous-sections

1. **[Button](./01-button.md)**
   - Variantes (primary, secondary, danger)
   - Tailles et états
   - Loading state

2. **[Input](./02-input.md)**
   - Text input avec validation
   - Password input
   - Number input

3. **[Table](./03-table.md)**
   - Tableau avec tri
   - Sélection de lignes
   - Pagination

4. **[Sidebar](./04-sidebar.md)**
   - Navigation sidebar
   - Badges et notifications
   - Collapsible

5. **[Modals](./05-modals.md)**
   - Dialogues modaux
   - Confirmation dialogs
   - Form modals

6. **[Toasts](./06-toasts.md)**
   - Notifications toast
   - Types (success, error, info)
   - Auto-dismiss

7. **[Inspector panels](./07-inspector-panels.md)**
   - Panneaux de debug
   - Inspection de l'état
   - Outils développeur

8. **[Command palette](./08-command-palette.md)**
   - Palette de commandes
   - Recherche fuzzy
   - Raccourcis clavier

## Objectifs d'apprentissage

À la fin de ce chapitre, vous serez capable de :

- Créer une bibliothèque de composants réutilisables
- Implémenter des composants complexes
- Construire des patterns UI professionnels
